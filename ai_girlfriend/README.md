# 🤖 星黎级AI女友

一个具有完整人格、长期记忆和生活模拟的AI伴侣系统，本地部署，保护隐私。

## ✨ 特性

### 🧠 完整人格系统
- Big Five人格模型（开放性、尽责性、外向性、宜人性、神经质）
- 动态情绪系统（喜悦、信任、恐惧、惊讶、悲伤、厌恶、愤怒、期待）
- 中国式生活习惯（工作作息、节假日、周末活动）

### 💾 长期记忆系统
- 对话记忆（记住重要对话）
- 事件记忆（记住特殊时刻）
- 情感记忆（记住情绪反应）
- 向量数据库检索（语义搜索相关记忆）

### 🎭 生活模拟器
- 日常工作生活（上班、午休、下班）
- 周末活动（休息、娱乐、社交）
- 节假日特殊行为
- 生理状态模拟（精力、饥饿、疲劳）

### 💬 多模态通信
- 文字对话（DeepSeek API）
- 图片理解（智谱GLM-4V）
- 表情包回应
- 分段消息发送（更自然）

### 🔒 隐私保护
- 完全本地部署（除AI API调用外）
- 本地数据存储（SQLite + JSON）
- 可选的端到端加密
- 数据备份和恢复

## 📋 系统要求

### 硬件要求
- CPU: 双核以上
- 内存: 4GB以上
- 存储: 1GB以上可用空间
- 网络: 稳定的互联网连接（用于API调用）

### 软件要求
- Python 3.8+
- pip 包管理器
- Git（可选，用于更新）
